Dependencies
```
Install 7Zip

```

To Install
```bash
git clone https://github.com/blh0021/winrb.git %USERPROFILE%\.winrb
```

Download a version of Ruby
```bash
rbenv install 2.1.5
```

To show install versions of Ruby
```bash
rbenv versions
```

To use a version of Ruby
```bash
rbenv local 2.1.5
```
